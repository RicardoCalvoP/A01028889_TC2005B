Anchor

The anchor tag is usually to connect different pages with the usage of Hyperlinks. You can also connect different parts of the same documents through id's in sections.
The anchor tags are: "<a> </a>"

Example

Esto no es un Rick Roll >:)

HTML Code

<a href="https://youtu.be/wAMZ6KpMGQI?t=7">Esto no es un Rick Roll >:) </a>

Paragraph

The paragraph tag is used to contain the text that is not a title or subtitle, just like this or any other block of text that adds information to the page. Some characteristics you need to know is that browsers will automatically add a single blank line before and after each tag element
Paragraph tag is written this way "<p> </p>"

Example

This paragraph was made with a paragraph tag

HTML Code

<p> This paragraph was made with a paragraph tag </p>

Headers

H2

The h2 tag is used to define headings of secondary importance. It represents a subheading or a section title.
H2 tag is written this way "<h2> </h2>"

Example

This is an H2 heading

HTML Code

<h2> This is an H2 heading </h2>

H3

This is an H3 heading

<h3> This is an H3 heading </h3>

Div

The div tag is a generic container used to divide the HTML document into sections. This tags are mostly used to later to when you apply css code into the page. Is good practice to have identifiers between <div> tags using the "class" attribute this way you can call every section different and work on them separately.
Div tag is written this way "<div class="class_name"> </div>"

Example

This content is inside a div container called "div_example".

HTML Code

<div class="div_example"> This content is inside a div container. </div>

Span

The span tag is an inline container used to mark up a part of a text or a part of a document. It is mostly similar to de <div> tag, the difference between both is that <span> works with inline structure while <div> works in more of a "blocky" way. <span> tag is also used to be modified using css code
Span tag is written this way "<span> </span>"

Example

This is a span element inside a sentence.

HTML Code

This is a <span>span</span> element inside a paragraph.
Later on we could change the contents color or maybe even highlight it.

Underline

The underline tag is used to underline text, although it's not recommended in modern web design. It is common to see misspelled words underline with this tags; although you can to the same with a span tag and modify it later with css tag.
Underline tag is written this way "<u> </u>"

Example

This is some mispeled text.

HTML Code

This is some <u>mispeled</u>; in a sentence

Bold

The bold tag is used to make text bold, although it's not recommended in modern web design. Just as the last tag, you can use a span tag and modify it later with css code. Also in difference between bold tag and strong, is that bold doesn't add any importance to the text, it just make is bulkier.
Bold tag is written this way "<b> </b>"

Example

This is a normal text | This is a bold text

HTML Code

This is a normal text | <b>This is a bold text</b>

Strike

The strike tag is used to draw a horizontal line through text, typically to indicate a deletion or strikeout.
The <strike> </strike> tag is unable in HTML 5; use instead <s> </s> for same results

Example

This sentence is strikeout

HTML Code

<s>This sentence is strikeout</s>

Header

The header tag is used to represent a container for a introduction where you can Normally find H1 to H2, logo, authorship, etc.
You can have more than one header tag, the only restrictions, you can't place a header tag inside a footer, address, or another header tag.

Example

This is a h1 inside a header

This text created by Rick is inside a header

HTML Code

<header>
<h1>This is a h1 inside a header</h1>
<p>This text created by Rick is inside a header</p>
</header>

Section

The section tag is used to define sections in a document, such as chapters, headers, footers, or any other sections of the document.
You can add an id attribute to interact later on with the section and differentiate one section from another

Example

This text is inside a section with id = Example

HTML Code

<Section id="Example">
<p> This text is inside a section with id = Example </p>
</Section>

It doesn't affect directly to the html but you can style it with css code.

Example

Article Example

This text is inside an article tag

Aside

The aside tag is used to define content aside from the content it is placed in, often represented as a sidebar or a pull-quote. Just as the last tag, the Aside tag doesn't render as something special in html, but you can style it later with css code.

Example

This a normal text in a paragraph

Aside Text

This text is inside an aside tag and it adds spaces between each tag automatically.

HTML Code

<p> This is a normal text in a paragraph. </p>
<Aside>
<h4> Aside Text </h4>
<p> This is inside a side an aside tag and it adds spaces between each tag automatically. </p>
</Aside>

Footer

Similar to the header tag, the footer tag defines a footer in a document or section, also you can have multiple footer tags in the same document.
The footer tag Normally contains authorship, copyright, contact information, sitemap, top links, related documents, etc.

Example

This text is inside a footer tag

Click here to go back to index

HTML Code

<footer>
<p>
This text is inside a footer tag
</p>
<p> <a href="#Index"> Click here to go back to index </a> </p>
</footer>

Navigation

The navigation tag is used to define a set of navigation links. Not all links need to be inside a nav tag, but it is usually used only for a major block of navigation links.

Example

  • Index
  • Navigation
  • Copyright

HTML Code

<nav>
<ul>
<li><a href="#home">Home</a></li>
<li><a href="#about">About</a></li>
<li><a href="#contact">Contact</a></li>
</ul>
</nav>

<br>
This is another paragraph separated by a break
</p>

Form

The form tag is used to create an HTML form for user input. Normally used to get Names, emails, phone numbers, etc.
You can differentiate different inputs with inputs types such as text, submit, mail, etc.

Example





HTML Code

<form>
<label for="fname">First name:</label>
<input type="text" id="fname" name="fname"><br><br>
<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname"><br><br>
<input type="submit" value="Submit">
</form>
