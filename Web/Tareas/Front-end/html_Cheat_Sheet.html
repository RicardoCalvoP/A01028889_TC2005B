Anchor

The anchor tag The anchor tags are: "<a> </a>"

Paragraph

The paragraph tag is used to contain the text that is not a title or subtitle, just like this or any other block of text that adds information to the page. Some characteristics you need to know is that browsers will automatically add a single blank line before and after each tag element
Paragraph tag is written this way "<p> </p>"

Example

This paragraph was made with a paragraph tag

HTML Code

<p> This paragraph was made with a paragraph tag </p>

Headers

H2

The h2 tag is used to define headings of secondary importance. It represents a subheading or a section title.
H2 tag is written this way "<h2> </h2>"

Example

This is an H2 heading

HTML Code

<h2> This is an H2 heading </h2>

H3

The h3 tag is used to define headings of tertiary importance. It represents a sub-subheading or a subsection title.
H3 tag is written this way "<h3> </h3>"

Example

This is an H3 heading

HTML Code

<h3> This is an H3 heading </h3>

H4

The h4 tag is used to define headings of quaternary importance. It represents a sub-sub-subheading or a subsection of a subsection title.
H4 tag is written this way "<h4> </h4>"

Example

This is an H4 heading

HTML Code

<h4> This is an H4 heading </h4>

The h5 tag is used to define headings of quinary importance. It represents a sub-sub-sub-subheading or a subsection of a subsection of a subsection title.
H5 tag is written this way "<h5> </h5>"

H6

The h6 tag is used to define headings of senary importance. It represents a sub-sub-sub-sub-subheading or a subsection of a subsection of a subsection of a subsection title.
H6 tag is written this way "<h6> </h6>"

Div

The div tag is a generic container used to divide the HTML document into sections. This tags are mostly used to later to when you apply css code into the page. Is good practice to have identifiers between <div> tags using the "class" attribute this way you can call every section different and work on them separately.
Div tag is written this way "<div class="class_name"> </div>"

Example

This content is inside a div container called "div_example".

HTML Code

<div class="div_example"> This content is inside a div container. </div>

Span

The span tag is an inline container used to mark up a part of a text or a part of a document. It is mostly similar to de <div> tag, the difference between both is that <span> works with inline structure while <div> works in more of a "blocky" way. <span> tag is also used to be modified using css code
Span tag is written this way "<span> </span>"

Example

This is a span element inside a sentence.

HTML Code

This is a <span>span</span> element inside a paragraph.
Later on we could change the contents color or maybe even highlight it.

Strong

The strong tag is used to indicate strong importance, typically displayed in bold.
Strong tag is written this way "<strong> </strong>"

Example

This is a very important word inside this line.

HTML Code

This is a very <strong>important</strong> word inside this line.

Italic

The italic tag is used to emphasize text, typically displayed in italic. This is often used when you want to phrase or use technical terms.
Italic tag is written this way "<i> </i>"

Example

This is an example using italics in a sentence.

HTML Code

Underline

The underline tag is used to underline text, although it's not recommended in modern web design. It is common to see misspelled words underline with this tags; although you can to the same with a span tag and modify it later with css tag.
Underline tag is written this way "<u> </u>"

Example

This is some mispeled text.

HTML Code

This is some <u>mispeled</u>; in a sentence

Bold

The bold tag is used to make text bold, although it's not recommended in modern web design. Just as the last tag, you can use a span tag and modify it later with css code. Also in difference between bold tag and strong, is that bold doesn't add any importance to the text, it just make is bulkier.
Bold tag is written this way "<b> </b>"

Example

This is a normal text | This is a bold text

HTML Code

This is a normal text | "<b>This is a bold text</b>"

Strike

The strike tag is used to draw a horizontal line through text, typically to indicate a deletion or strikeout.

Small

The small tag is used to represent small print or fine print.

Emphasized

The emphasized tag is used to emphasize text, typically displayed in italic.

Header

The header tag is used to define a header for a document or a section.

Section

The section tag is used to define sections in a document, such as chapters, headers, footers, or any other sections of the document.

Article

The article tag is used to define independent, self-contained content that could be distributed and reused independently.

Aside

The aside tag is used to define content aside from the content it is placed in, often represented as a sidebar or a pull-quote.

Footer

The footer tag is used to define a footer for a document or a section.

Navigation

The navigation tag is used to define a set of navigation links.

Image

The image tag is used to embed images in an HTML document.

Break

The break tag is used to insert a line break in the document.

Horizontal Rule

The horizontal rule tag is used to create a thematic break in the document, typically displayed as a horizontal line.

Form

The form tag is used to create an HTML form for user input.

Table

The table tag is used to create a table in an HTML document.

Unordered List

The unordered list tag is used to create an unordered list in an HTML document.

Ordered List

The ordered list tag is used to create an ordered list in an HTML document.
